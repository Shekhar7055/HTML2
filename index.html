Id Attribute is used to specify a unique id for an html element. You cannot have more than one element with the same id in an html document. id must be unique within the html element.

Syntax for id -> write a period(#) character, followed by a id name in a style sheet, then define a css property within a curly braces {}.

Difference between id and class -> A class name can be used in multiple html element, while an id must only be used by one html element within the page.
